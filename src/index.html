HTML

CSS

JavaScript

Git

A distributed version control system that allows you to track changes in files and work on the same project together with colleagues

OOP

A methodology in which the entire program is viewed as a set of interacting objects. At the same time, it is important for us to know their characteristics

JSON

JavaScript-based data exchange text format. But at the same time, the format does not depend on JavaScript and can be used in any programming language
